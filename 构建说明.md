# 无忧辅助 - 构建指南

## 项目结构

```
无忧辅助/
├── 无忧辅助/              # Swift 源码 (UI 层)
│   ├── 无忧辅助App.swift       # App 入口
│   ├── MainTabView.swift       # Tab 导航
│   ├── PhoneControlView.swift  # 【手机控制区域】重启/注销
│   ├── ScriptControlView.swift # 脚本控制 (预留)
│   ├── LogView.swift           # 运行日志
│   ├── RootHelper.swift        # posix_spawn root 调用
│   ├── Log.swift               # 日志管理
│   └── Info.plist              # App 配置
├── roothelper/              # C 内核提权工具
│   ├── main.c               #   入口 + 命令分发 (reboot/escalate/respring)
│   ├── kfd.c / kfd.h        #   内核提权: task_for_pid(0) + ucred 修改
│   └── offsets.c / offsets.h #   内核偏移表 (iOS 14-17)
├── entitlements.plist          # 全部权限 (TrollStore)
├── project.yml                 # XcodeGen 配置
├── build_helper.sh             # 编译 C helper
├── build.sh                    # 一键构建 IPA
└── exportOptions.plist         # 导出配置
```

## 构建步骤

### 方式一: XcodeGen + Xcode (推荐)

```bash
# 1. 安装 XcodeGen
brew install xcodegen

# 2. 生成 Xcode 项目
cd 无忧辅助
xcodegen generate

# 3. 编译 C helper
bash build_helper.sh

# 4. 打开 Xcode 项目
open 无忧辅助.xcodeproj

# 5. 在 Xcode 中:
#    - Product → Archive
#    - 导出 .app
#    - 或直接 Build（产物在 DerivedData）
```

### 方式二: 手动创建 Xcode 项目

1. 打开 Xcode → New Project → iOS App
2. 名称: `无忧辅助`, Bundle ID: `com.wuyou.helper`
3. 语言: Swift, Interface: SwiftUI
4. 把所有 `无忧辅助/*.swift` 文件加入项目
5. 在 Build Phases → Copy Bundle Resources 中添加 `roothelper` 二进制
6. 设置 Code Signing Identity 为 `-` (Ad Hoc)
7. Build 并导出 IPA

### 方式三: 命令行一键构建

```bash
# 完整构建 (需要先有 Xcode 项目)
bash build.sh
```

## 权限说明

| 权限 Key | 用途 |
|---------|------|
| `no-sandbox` | 彻底关闭沙盒 |
| `platform-application` | 平台级应用 |
| `get-task-allow` | task_for_pid 权限 |
| `com.apple.system-task-ports` | 获取 kernel_task 端口 |
| `com.apple.private.security.system-actions` | 重启/关机系统命令 |
| `dynamic-codesigning` | 允许 ldid 伪签名生效 |
| `com.apple.private.security.no-entitlements` | 跳过 entitlement 检查 |

### 设计决策记录

- **CI: `macos-15` 锁定** — 不用 `macos-latest`，避免每半年修一次 SDK 移除 API。巨魔仅需 iOS 14-17 SDK。
- **部署目标: iOS 14.0** — TrollStore 最低支持版本，因此用 `NavigationView`/`accentColor`/`PreviewProvider` 而非 iOS 16+ 新 API。
- **提权方式: `task_for_pid(0)`** — TrollStore 签名自带 `system-task-ports`，可直接拿 kernel_task 端口做内存读写，无需 phypuppet/smith 等内核漏洞。
- **`@_silgen_name`** — 用编译期链接替代 `dlsym`，调用未公开的 `proc_listpids`/`proc_name` 函数。

## 安装 (TrollStore)

1. 导出 IPA 后传到手机
2. 用 TrollStore 打开 IPA 安装
3. 打开「无忧辅助」App
4. Tab 1「手机控制」→ 点击「重启手机」或「注销手机」
